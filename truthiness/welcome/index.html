You're all set — Fact Checker

Thanks for installing! Fact Checker is a free fact checker built right into your right-click menu. Select any text — a sentence, a paragraph, an AI answer, a viral post — and it will fact-check each claim: verify it or debunk it, true or false, and flag the misinformation that polished wording can hide.

Fact-check anything in 10 seconds

  1. Select the text you want to check

    Highlight a claim, a sentence, or a whole paragraph on any web page — a news story, an AI answer, a social post.

  2. Right-click → “Fact-check selection”

    Choose Fact-check selection from the right-click menu. That's the whole interface — there's nothing to set up, no account, and no API key.

  3. Read the per-claim verdicts

    A panel opens on the page with each distinct claim, a color-coded verdict, the reasoning, any omitted nuance, and the sources the check relied on.

What each verdict means

  • accurate
  • partially accurate
  • oversimplified
  • misleading
  • unreliable
  • inaccurate
  • unverifiable
  • unfalsifiable

Free & private

  • Completely free — no account, no sign-up, no API key.
  • Forward-and-forget — your selected text is sent only to run the check and is never stored on our server.
  • Minimal by design — to help judge a claim, the check also sees the page's domain and title (never the full URL), and nothing for local or private pages.
  • No tracking — no analytics, no ads, no browsing history, no selling your data.
