Fact Checker — Privacy Policy

Publisher: Trimtab Labs · Effective date: 9 July 2026
Applies to the Fact Checker Chrome extension (internal name "Truthiness").

Fact Checker fact-checks text you explicitly select. It is completely free — no account, no sign-up, and no API key — and it is built to collect as little as possible and to forward and forget: nothing you check is stored on our server.

What gets sent, and where

A fact-check runs only when you select text and choose “Fact-check selection.” When you do, the following is sent through our own proxy to Google Gemini, which returns the per-claim verdicts:

  • The text you selected. Only that selection — never the whole page.
  • Light page context: the site’s domain and the page title (for example, example.com and “Example headline”). This helps the model understand what a claim refers to. We send only the domain and title — never the full URL, path, or query string — and we do not send any context for chrome://, file://, local, or private/intranet pages.

Our proxy does not store, log, or retain the selected text, the domain, or the title — it forwards them to run the check and keeps only anonymous aggregate counts (for rate-limiting and totals). Because this is a free service, the AI provider (Google) may use submitted content to improve its models under its own terms.

What we store

On our server: a random install identifier (used only for rate-limiting and abuse prevention) and anonymous aggregate counts. We do not store your email, the text you check, or the pages you visit. In your browser: only that random install identifier (chrome.storage.local). There are no settings to save — the extension has no options to configure.

What we do not do

No accounts, no API keys, no payments. No analytics, no tracking, no advertising, no selling or sharing of data, and no access to your browsing history. The extension reads page content only for the specific selection you fact-check, only when you trigger it.

Permissions

contextMenus (the right-click action), storage (the random install id), activeTab + scripting (to read your selection and draw the results panel on the page when you click), and host access limited to our own proxy (truthiness-proxy.fly.dev), used only to run a check you initiated.

Limited Use

Fact Checker’s use of information complies with the Chrome Web Store User Data Policy, including the Limited Use requirements.
